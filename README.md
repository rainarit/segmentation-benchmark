# Image Segmentation Benchmark Documentation
## Prerequisites
Install Git Repository
```bash
git clone https://github.com/rainarit/segmentation_benchmark.git
cd segmentation_benchmark
```
## How to Run:
### Download COCO Dataset:
```bash
./coco_download.sh
```
### Run Training Script:
```bash
./train_fcn_resnet50_coco.sh
```
